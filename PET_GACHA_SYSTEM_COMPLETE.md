## 🎮 Gameplay Flow

### 1. First Time User
```
1. Buka /gacha-pet
2. Tab "Pet" = KOSONG! 🥚
3. Lihat "Belum Punya Pet!"
4. Klik "Buka Gacha Sekarang!"
```

### 2. Gacha Pull
```
1. Tab "Gacha"
2. Pull 1x (100 coins) atau 10x (1000 coins)
3. Dapat item + kemungkinan Pet Egg!
4. Jika dapat Pet Egg → UNLOCK PET!
```

### 3. Pet Unlock
```
1. Pull dapat "Cat Egg"
2. Auto unlock Cat!
3. Notifikasi: "Dapat Cat!"
4. Tab "Pet" sekarang ada Cat
5. Bisa ganti expression
```

### 4. Duplicate Protection
```
1. Sudah punya Cat
2. Pull dapat "Cat Egg" lagi
3. Auto jadi 200 coins compensation
4. Chat: "Duplicate! +200 coins"
```

### 5. Collection
```
1. Buka tab "Pet"
2. Lihat semua pet yang dimiliki
3. Klik pet untuk select
4. SSR pets punya special glow!
```
