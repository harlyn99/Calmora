# 🚀 Ide Fitur Berguna & Seru untuk Calmora

## 🎮 Gamification & Motivation

### 1. **Achievement System** 🏆
```
Badge yang bisa dibuka:
- 🌅 Early Bird: Complete 5 morning sessions
- 🔥 7-Day Streak: Use app 7 days straight
- 💰 Coin Master: Earn 500 coins
- 📚 Knowledge Seeker: Complete 20 learning tasks
- 🧹 Clean Freak: Finish all chores 10x
- 🎯 Focus Master: 1000 focus minutes
- 🌟 Perfect Day: Complete all schedule tasks
```

**Implementasi:** Modal popup saat dapat badge + collectible gallery

---

### 2. **Level & XP System** ⭐
```
User Level berdasarkan:
-完成任务 = +XP
- Focus sessions = +XP
- Daily streak = +XP bonus

Level Up rewards:
- Level 5: Unlock new pet
- Level 10: Unlock furniture set
- Level 20: Unlock special theme
- Level 50: Legendary badge
```

**Progress Bar** di dashboard dengan next level milestone

---

### 3. **Daily Quest/Missions** 📜
```
Quest Harian (Random 3 per day):
- [ ] Complete 3 HIGH energy tasks → +50 coins
- [ ] 100 focus minutes → +30 coins
- [ ] Feed pet 3x → +20 coins
- [ ] Complete all chores → +40 coins
- [ ] No skipped tasks → +25 coins

Quest Weekly:
- [ ] 7-day streak → 200 coins + exclusive item
- [ ] 500 focus minutes → 150 coins
```

**UI:** Quest board di dashboard dengan checkmarks

---

### 4. **Pet Evolution System** 🐾
```
Pet evolves based on care & level:

Bunny:
  Baby → Teen → Adult → Legendary
  
Cat:
  Kitten → Cool Cat → Wise Cat → Cosmic Cat
  
Bear:
  Cub → Strong Bear → Elder Bear → Spirit Bear

Evolution requirements:
- Level 10 + Happiness 90% + 30 days care
```

**Visual:** Pet appearance changes dengan evolution

---

### 5. **Pet Mini Games** 🎯
```
Quick games (30-60 seconds):
- 🎮 Catch falling coins
- 🎮 Memory card match
- 🎮 Whack-a-mole
- 🎮 Feed pet correctly
- 🎮 Dress up challenge

Rewards:
- Win = coins + happiness boost
- High score = bonus XP
```

**Leaderboard:** Weekly high scores
